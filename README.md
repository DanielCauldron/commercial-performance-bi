## 🧾 Descrição do Projeto — Dashboard de Vendas

<img width="1772" height="794" alt="Captura de tela 2026-01-08 235238" src="https://github.com/user-attachments/assets/fbdcca39-5a8f-48d6-9a65-6054b3064ba3" />


Este projeto desenvolve um dashboard analítico para monitoramento de desempenho comercial, com foco no volume de vendas, número de pedidos e participação dos produtos no faturamento total.

A solução foi construída em Power BI utilizando um modelo dimensional com tabelas fato e dimensão, medidas DAX e KPIs executivos. Essa modelagem permite escalabilidade e flexibilidade para análises futuras sem necessidade de reestruturação do modelo.

O painel facilita a identificação de produtos mais relevantes, comportamentos de compra e distribuição de faturamento, habilitando decisões estratégicas relacionadas a mix, priorização de estoque e política comercial.

## 🔧 Tecnologias e técnicas utilizadas

Power BI

DAX (medidas e KPIs)

Modelagem Dimensional (Fato + Dimensões)

Boas práticas de UX aplicado a BI

Storytelling orientado a decisão

Limpeza e padronização de dados

## 📊 Destaques técnicos

Construção de medidas: Total Vendas, Total Pedidos, Ticket Médio, Participação %

Transformações: agregações, tipagem, padronização e hierarquização

Visualização com hierarquia analítica: KPIs → ranking → detalhamento

Dashboard responsivo e com alta densidade informacional

## 🎯 Resultado

O usuário final consegue rapidamente identificar quais produtos impulsionam o faturamento e como estão distribuídos os pedidos, reduzindo tempo de análise e aumentando a assertividade na tomada de decisão comercial.
